## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add textarea
```



## Usage

```ts
import '@bootstrap-wc/components/textarea';
```

```html
<bs-textarea placeholder="Your message" rows="4"></bs-textarea>
```

See `packages/components/src/textarea/` in the source for the complete property / event list.
