Transient notification toast.

## Example

<Example code={`<bs-toast open heading="Hello">This is a toast.</bs-toast>`} />

## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add toast
```



## Usage

```ts
import '@bootstrap-wc/components/toast';
```

```html
<bs-toast open heading="Hello">This is a toast.</bs-toast>
```

See `packages/components/src/toast/` in the source for the complete property / event list.
