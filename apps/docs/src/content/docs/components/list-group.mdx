A flexible list of content. The host `<bs-list-group>` carries `.list-group`
and each `<bs-list-group-item>` carries `.list-group-item`, so Bootstrap's
sibling / child selectors apply across the slot boundary without any extra
wrapper.

## Basic example

<Example code={`<bs-list-group>
  <bs-list-group-item>An item</bs-list-group-item>
  <bs-list-group-item>A second item</bs-list-group-item>
  <bs-list-group-item>A third item</bs-list-group-item>
  <bs-list-group-item>A fourth item</bs-list-group-item>
  <bs-list-group-item>And a fifth one</bs-list-group-item>
</bs-list-group>`} />

## Active items

Add the `active` attribute to a `<bs-list-group-item>` to mark the current
selection. The host reflects `aria-current="true"`.

<Example code={`<bs-list-group>
  <bs-list-group-item active>An active item</bs-list-group-item>
  <bs-list-group-item>A second item</bs-list-group-item>
  <bs-list-group-item>A third item</bs-list-group-item>
  <bs-list-group-item>A fourth item</bs-list-group-item>
  <bs-list-group-item>And a fifth one</bs-list-group-item>
</bs-list-group>`} />

## Links and buttons

Set `href` to make an item behave as a link, or `action` to give it the
hover / focus treatment without a URL. Both variants receive the
`.list-group-item-action` host class.

<Example code={`<bs-list-group>
  <bs-list-group-item href="#" active>The current link item</bs-list-group-item>
  <bs-list-group-item href="#">A second link item</bs-list-group-item>
  <bs-list-group-item href="#">A third link item</bs-list-group-item>
  <bs-list-group-item href="#">A fourth link item</bs-list-group-item>
  <bs-list-group-item href="#" disabled>A disabled link item</bs-list-group-item>
</bs-list-group>`} />

<Example code={`<bs-list-group>
  <bs-list-group-item action active>The current button</bs-list-group-item>
  <bs-list-group-item action>A second button item</bs-list-group-item>
  <bs-list-group-item action>A third button item</bs-list-group-item>
  <bs-list-group-item action>A fourth button item</bs-list-group-item>
  <bs-list-group-item action disabled>A disabled button item</bs-list-group-item>
</bs-list-group>`} />

## JavaScript behavior

A list group can drive its own tab panels — use `<bs-list-group-item action>`
for each row and toggle `.active` / `.show` on matching `<div class="tab-pane">`
elements. The markup is identical to Bootstrap's native pattern, minus the
`data-bs-toggle` attribute (web-component consumers wire their own click
handler).

<Example code={`<div class="row">
  <div class="col-4">
    <bs-list-group role="tablist">
      <bs-list-group-item action active role="tab" aria-controls="list-home" aria-selected="true">Home</bs-list-group-item>
      <bs-list-group-item action role="tab" aria-controls="list-profile" aria-selected="false">Profile</bs-list-group-item>
      <bs-list-group-item action role="tab" aria-controls="list-messages" aria-selected="false">Messages</bs-list-group-item>
      <bs-list-group-item action role="tab" aria-controls="list-settings" aria-selected="false">Settings</bs-list-group-item>
    </bs-list-group>
  </div>
  <div class="col-8">
    <div class="tab-content">
      <div class="tab-pane fade show active" id="list-home" role="tabpanel">
        <p>Some placeholder content in a paragraph relating to "Home". In production, you would obviously have more real content here.</p>
      </div>
      <div class="tab-pane fade" id="list-profile" role="tabpanel">
        <p>Some placeholder content in a paragraph relating to "Profile".</p>
      </div>
      <div class="tab-pane fade" id="list-messages" role="tabpanel">
        <p>Some placeholder content in a paragraph relating to "Messages".</p>
      </div>
      <div class="tab-pane fade" id="list-settings" role="tabpanel">
        <p>Some placeholder content in a paragraph relating to "Settings".</p>
      </div>
    </div>
  </div>
</div>`} />

## API

### `<bs-list-group>` attributes / properties

| Name | Type | Default | Description |
|---|---|---|---|
| `flush` | `boolean` | `false` | Removes outer borders and rounded corners. |
| `numbered` | `boolean` | `false` | Renders numeric counters via `::before` on each item. |
| `horizontal` | `boolean` \| `sm` \| `md` \| `lg` \| `xl` \| `xxl` | — | Lay items in a row. Pass a breakpoint to only go horizontal above it. |

### `<bs-list-group-item>` attributes / properties

| Name | Type | Default | Description |
|---|---|---|---|
| `active` | `boolean` | `false` | Applies the `.active` state and sets `aria-current="true"`. |
| `disabled` | `boolean` | `false` | Styles as disabled and suppresses click navigation. Sets `aria-disabled="true"`. |
| `variant` | `primary` \| `secondary` \| `success` \| `danger` \| `warning` \| `info` \| `light` \| `dark` | — | Bootstrap contextual color. |
| `href` | `string` | — | When set, clicking the item navigates to the URL. Also adds `.list-group-item-action`. |
| `action` | `boolean` | `false` | Adds `.list-group-item-action` for the hover / focus treatment without a URL. |

```sh
# as an npm package

```

```html
  <bs-list-group-item disabled>Disabled</bs-list-group-item>
```
