## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add card
```



## Usage

```ts

```

```html
<bs-card heading="Card title" subtitle="Subtitle">Some quick example text.</bs-card>
```

See `packages/components/src/card/` in the source for the complete property / event list.
