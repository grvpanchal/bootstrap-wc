## Example

<Example code={`<bs-progress value="65" variant="info"></bs-progress>`} />

## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add progress
```



## Usage

```ts
import '@bootstrap-wc/components/progress';
```

```html
<bs-progress value="65" variant="info"></bs-progress>
```

See `packages/components/src/progress/` in the source for the complete property / event list.
