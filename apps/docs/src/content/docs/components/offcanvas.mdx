## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add offcanvas
```

Additional peer deps: `focus-trap`

## Usage

```ts
import '@bootstrap-wc/components/offcanvas';
```

```html
<bs-button>Open</bs-button>
<bs-offcanvas heading="Sidebar">Offcanvas content.</bs-offcanvas>
```

See `packages/components/src/offcanvas/` in the source for the complete property / event list.
