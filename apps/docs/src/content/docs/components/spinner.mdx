`<bs-spinner>` renders a Bootstrap loading spinner. Both the border and grow
animations are supported, along with every theme color and a small-size
modifier. Spinners are `role="status"` by default and ship a visually-hidden
label for assistive technologies.

## Border spinner

<Example code={`<bs-spinner></bs-spinner>`} />

Use any of Bootstrap's margin utilities on the host to control spacing.

<Example code={`<bs-spinner class="m-5"></bs-spinner>`} />

<Example code={`<div class="clearfix">

<Example code={`<div class="text-center">

## Size

Use `size="sm"` for a compact spinner.

<Example code={`<bs-spinner size="sm"></bs-spinner>
<bs-spinner type="grow" size="sm"></bs-spinner>`} />

For a custom size, set the `width` and `height` attributes.

<Example code={`<bs-spinner width="3rem" height="3rem"></bs-spinner>
<bs-spinner type="grow" width="3rem" height="3rem"></bs-spinner>`} />

## Buttons

Drop a small spinner inside a `<bs-button>` to indicate an in-progress action.

<Example code={`<bs-button variant="primary" disabled>
  <bs-spinner size="sm" aria-hidden="true"></bs-spinner>
  <span class="visually-hidden" role="status">Loading...</span>
</bs-button>
<bs-button variant="primary" disabled>

## API

### Attributes / properties

| Name | Type | Default | Description |
|---|---|---|---|
| `type` | `border` \| `grow` | `border` | Animation style. |
| `variant` | `primary` \| `secondary` \| `success` \| `danger` \| `warning` \| `info` \| `light` \| `dark` | — | Bootstrap theme color applied via `text-{variant}`. If omitted the spinner inherits the surrounding `currentColor` — the preferred choice inside a colored button. |
| `size` | `sm` \| `md` | `md` | Size modifier. `sm` applies the `spinner-{type}-sm` class. |
| `width` | `string` | — | Custom width (any CSS length) for the spinner. Overrides the `size` modifier. |
| `height` | `string` | — | Custom height (any CSS length) for the spinner. Overrides the `size` modifier. |
| `label` | `string` | `Loading...` | Visually hidden text announced to assistive technologies. |

### Shadow parts

| Part | Description |
|---|---|
| `spinner` | The inner spinner element. |

## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add spinner
```

## Usage

```ts
import '@bootstrap-wc/components/spinner';
```

```html
<bs-spinner variant="primary"></bs-spinner>
```
