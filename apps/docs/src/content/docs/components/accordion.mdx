Expandable accordion with single- or multi-open behavior.

## Example

<Example code={`<bs-accordion>
  <bs-accordion-item heading="Item #1" open>First panel content.</bs-accordion-item>
  <bs-accordion-item heading="Item #2">Second panel content.</bs-accordion-item>
</bs-accordion>`} />

## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add accordion
```



## Usage

```ts
import '@bootstrap-wc/components/accordion';
```

```html
<bs-accordion>
  <bs-accordion-item heading="Item #1" open>First panel content.</bs-accordion-item>
  <bs-accordion-item heading="Item #2">Second panel content.</bs-accordion-item>
</bs-accordion>
```

See `packages/components/src/accordion/` in the source for the complete property / event list.
