Groups related buttons.

## Example

<Example code={`<bs-button-group><bs-button>Left</bs-button><bs-button>Middle</bs-button><bs-button>Right</bs-button></bs-button-group>`} />

## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add button-group
```



## Usage

```ts
import '@bootstrap-wc/components/button-group';
```

```html
<bs-button-group><bs-button>Left</bs-button><bs-button>Middle</bs-button><bs-button>Right</bs-button></bs-button-group>
```

See `packages/components/src/button-group/` in the source for the complete property / event list.
