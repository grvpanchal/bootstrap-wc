## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add form-check
```

```ts

```

```html
<bs-form-check label="Check me" type="checkbox"></bs-form-check>
<bs-form-check label="Switch on" type="switch"></bs-form-check>
```
