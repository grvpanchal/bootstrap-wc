## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add nav
```



## Usage

```ts
import '@bootstrap-wc/components/nav';
```

```html
<bs-nav nav-style="pills">
  <bs-nav-item active href="#">Active</bs-nav-item>
  <bs-nav-item href="#">Link</bs-nav-item>
  <bs-nav-item disabled href="#">Disabled</bs-nav-item>
</bs-nav>
```

See `packages/components/src/nav/` in the source for the complete property / event list.
