The `<bs-nav>` element carries Bootstrap's `.nav` class (plus the `.nav-tabs`,
`.nav-pills`, or `.nav-underline` modifiers) on its host. Slotted
`<bs-nav-item>` children carry `.nav-link` on their host, so Bootstrap's
descendant combinators (`.nav-tabs .nav-link.active`, `.nav-fill > .nav-link`,
etc.) resolve across the shadow boundary.

For dynamic, tabbed interfaces with panels and JavaScript-driven activation,
see [`bs-tabs`](/components/tabs/). This page covers the static nav styles.

<Example code={`<bs-nav class="justify-content-center">
  <bs-nav-item active href="#">Active</bs-nav-item>
  <bs-nav-item href="#">Link</bs-nav-item>
  <bs-nav-item href="#">Link</bs-nav-item>
  <bs-nav-item disabled href="#">Disabled</bs-nav-item>
</bs-nav>`} />

Right-align with `.justify-content-end`:

<Example code={`<bs-nav class="justify-content-end">
  <bs-nav-item active href="#">Active</bs-nav-item>

Use `nav-style="underline"` for an underlined-link style (Bootstrap 5.3+):

<Example code={`<bs-nav nav-style="underline">

## Fill and justify

Use `fill="fill"` to have items proportionally fill the horizontal space:

<Example code={`<bs-nav nav-style="pills" fill="fill">
  <bs-nav-item active href="#">Active</bs-nav-item>
  <bs-nav-item href="#">Much longer nav link</bs-nav-item>
  <bs-nav-item href="#">Link</bs-nav-item>
  <bs-nav-item disabled href="#">Disabled</bs-nav-item>
</bs-nav>`} />

Use `fill="justified"` for items of equal width:

<Example code={`<bs-nav nav-style="pills" fill="justified">
  <bs-nav-item active href="#">Active</bs-nav-item>
  <bs-nav-item href="#">Much longer nav link</bs-nav-item>
  <bs-nav-item href="#">Link</bs-nav-item>
  <bs-nav-item disabled href="#">Disabled</bs-nav-item>
</bs-nav>`} />

## Working with flex utilities

Combine with Bootstrap's flex utilities for responsive navs that stack on
small screens and go horizontal from the `sm` breakpoint up:

<Example code={`<bs-nav nav-style="pills" class="flex-column flex-sm-row">
  <bs-nav-item class="flex-sm-fill text-sm-center" active href="#">Active</bs-nav-item>
  <bs-nav-item class="flex-sm-fill text-sm-center" href="#">Longer nav link</bs-nav-item>
  <bs-nav-item class="flex-sm-fill text-sm-center" href="#">Link</bs-nav-item>
  <bs-nav-item class="flex-sm-fill text-sm-center" disabled href="#">Disabled</bs-nav-item>
</bs-nav>`} />

## API

### `<bs-nav>` attributes

| Name | Type | Default | Description |
|---|---|---|---|
| `nav-style` | `default` \| `tabs` \| `pills` \| `underline` | `default` | Visual style variant. |
| `fill` | `none` \| `fill` \| `justified` | `none` | How items distribute along the main axis. |
| `vertical` | `boolean` | `false` | Stacks items vertically via `.flex-column`. |

### `<bs-nav-item>` attributes

| Name | Type | Default | Description |
|---|---|---|---|
| `active` | `boolean` | `false` | Applies the `.active` state and `aria-selected="true"`. |
| `disabled` | `boolean` | `false` | Applies the `.disabled` state and `aria-disabled="true"`. |
| `href` | `string` | `#` | Navigation target on activation. |
| `controls` | `string` | — | Sets `aria-controls` for tab/panel wiring. |
