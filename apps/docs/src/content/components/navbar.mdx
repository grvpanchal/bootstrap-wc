Responsive navigation bar with brand, nav, forms, text and a collapse toggler.

## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add navbar
```

## Usage

```ts
import '@bootstrap-wc/components/navbar';
```

## Supported content

A responsive navbar that collapses at the `lg` breakpoint with brand, nav,
a dropdown and a search form.

<Example code={`<bs-navbar background="body-tertiary">
  <span slot="brand">Navbar</span>
  <bs-nav>
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Link</bs-nav-item>
    <bs-nav-item disabled>Disabled</bs-nav-item>
  </bs-nav>
  <form slot="end" class="d-flex" role="search">
    <bs-input type="search" placeholder="Search" aria-label="Search" class="me-2"></bs-input>
    <bs-button variant="success" button-style="outline" type="submit">Search</bs-button>
  </form>
</bs-navbar>`} />

## Brand

The brand is slotted via `slot="brand"` and automatically wrapped in an
anchor with `.navbar-brand`. The target is configured via `brand-href`.

### Text

<Example code={`<bs-navbar background="body-tertiary" expand="never">
  <span slot="brand">Navbar</span>
</bs-navbar>`} />

### Image

Drop an `<img>` into the brand slot for an image-only brand.

<Example code={`<bs-navbar background="body-tertiary" expand="never">
  <img slot="brand" src="https://getbootstrap.com/docs/5.3/assets/brand/bootstrap-logo.svg" alt="Bootstrap" width="30" height="24" />
</bs-navbar>`} />

### Image and text

<Example code={`<bs-navbar background="body-tertiary" expand="never">
  <span slot="brand" class="d-inline-flex align-items-center">
    <img src="https://getbootstrap.com/docs/5.3/assets/brand/bootstrap-logo.svg" alt="Bootstrap" width="30" height="24" class="d-inline-block align-text-top me-2" />
    Bootstrap
  </span>
</bs-navbar>`} />

## Nav

Use `<bs-nav>` with `<bs-nav-item>` children for the primary navigation.
Hosts carry `.nav-link`, `.active`, and `.disabled` so Bootstrap's selectors
apply across the shadow boundary.

<Example code={`<bs-navbar background="body-tertiary">
  <span slot="brand">Navbar</span>
  <bs-nav>
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Features</bs-nav-item>
    <bs-nav-item href="#">Pricing</bs-nav-item>
    <bs-nav-item disabled>Disabled</bs-nav-item>
  </bs-nav>
</bs-navbar>`} />

## Forms

Slot any `<form>` into the `end` slot (or as a default child) to place it
inside the collapsible content.

<Example code={`<bs-navbar background="body-tertiary">
  <span slot="brand">Navbar</span>
  <form slot="end" class="d-flex" role="search">
    <bs-input type="search" placeholder="Search" aria-label="Search" class="me-2"></bs-input>
    <bs-button variant="success" button-style="outline" type="submit">Search</bs-button>
  </form>
</bs-navbar>`} />

Input groups work too:

<Example code={`<bs-navbar background="body-tertiary">
  <span slot="brand">Navbar</span>
  <form slot="end" class="d-flex" role="search">
    <bs-input-group>
      <bs-input-text>@</bs-input-text>
      <bs-input type="text" placeholder="Username" aria-label="Username"></bs-input>
    </bs-input-group>
  </form>
</bs-navbar>`} />

## Text

`.navbar-text` aligns strings of text vertically inside the navbar.

<Example code={`<bs-navbar background="body-tertiary">
  <span class="navbar-text">Navbar text with an inline element</span>
</bs-navbar>`} />

Mix navs and text:

<Example code={`<bs-navbar background="body-tertiary">
  <span slot="brand">Navbar w/ text</span>
  <bs-nav>
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Features</bs-nav-item>
    <bs-nav-item href="#">Pricing</bs-nav-item>
  </bs-nav>
  <span slot="end" class="navbar-text">Navbar text with an inline element</span>
</bs-navbar>`} />

## Color schemes

Navbars inherit their color mode from the document by default. Set
`theme="dark"` to force dark mode (adds `data-bs-theme="dark"` to the
`.navbar`) and combine with `background="*"` utilities for the final hue.

<Example code={`<bs-navbar background="dark" theme="dark">
  <span slot="brand">Navbar</span>
  <bs-nav>
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Features</bs-nav-item>
    <bs-nav-item href="#">Pricing</bs-nav-item>
  </bs-nav>
</bs-navbar>`} />

<Example code={`<bs-navbar background="primary" theme="dark">
  <span slot="brand">Navbar</span>
  <bs-nav>
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Features</bs-nav-item>
    <bs-nav-item href="#">Pricing</bs-nav-item>
  </bs-nav>
</bs-navbar>`} />

<Example code={`<bs-navbar theme="light" style="background-color: #e3f2fd">
  <span slot="brand">Navbar</span>

    <bs-nav-item href="#">Features</bs-nav-item>

  </bs-nav>
</bs-navbar>`} />

## Containers

Set `container="sm|md|lg|xl|xxl"` to swap the default `.container-fluid`
for a responsive container. Use `container="none"` to skip the inner
wrapper entirely when you need to nest the navbar inside your own outer
`.container`.

<Example code={`<div class="container">
  <bs-navbar background="body-tertiary" container="none">
    <span slot="brand">Navbar</span>
  </bs-navbar>
</div>`} />

<Example code={`<bs-navbar background="body-tertiary" container="md">
  <span slot="brand">Navbar</span>
</bs-navbar>`} />

## Placement

The `placement` attribute pins the navbar to the viewport. Valid values
are `static` (default), `fixed-top`, `fixed-bottom`, `sticky-top` and
`sticky-bottom`. The legacy `fixed` / `sticky` booleans still work as
shorthands for `fixed-top` / `sticky-top`.

<Example code={`<bs-navbar background="body-tertiary">
  <span slot="brand">Default</span>
</bs-navbar>`} />

<Example code={`<bs-navbar background="body-tertiary" placement="fixed-top">
  <span slot="brand">Fixed top</span>
</bs-navbar>`} />

<Example code={`<bs-navbar background="body-tertiary" placement="fixed-bottom">
  <span slot="brand">Fixed bottom</span>
</bs-navbar>`} />

<Example code={`<bs-navbar background="body-tertiary" placement="sticky-top">
  <span slot="brand">Sticky top</span>
</bs-navbar>`} />

<Example code={`<bs-navbar background="body-tertiary" placement="sticky-bottom">
  <span slot="brand">Sticky bottom</span>
</bs-navbar>`} />

## Scrolling

Add the `.navbar-nav-scroll` class to the slotted `<bs-nav>` to cap its
height and scroll overflow.

<Example code={`<bs-navbar background="body-tertiary">
  <span slot="brand">Navbar scroll</span>
  <bs-nav class="navbar-nav-scroll" style="--bs-scroll-height: 100px;">
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Link</bs-nav-item>
    <bs-nav-item href="#">Link</bs-nav-item>
    <bs-nav-item href="#">Link</bs-nav-item>
    <bs-nav-item href="#">Link</bs-nav-item>
  </bs-nav>
</bs-navbar>`} />

## Responsive behaviors

Choose the breakpoint at which the navbar collapses with `expand`.
Acceptable values are `sm`, `md`, `lg` (default), `xl`, `xxl`, `always`
(never collapses) and `never` (always collapsed).

<Example code={`<bs-navbar background="body-tertiary" expand="sm">
  <span slot="brand">Expand sm</span>
  <bs-nav>
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Features</bs-nav-item>
  </bs-nav>
</bs-navbar>`} />

<Example code={`<bs-navbar background="body-tertiary" expand="always">
  <span slot="brand">Always expanded</span>
  <bs-nav>
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Features</bs-nav-item>
  </bs-nav>
</bs-navbar>`} />

### Toggler

By default the toggler sits to the right. Reorder via the standard
`order-*` utilities on slotted content. The built-in toggler icon uses
the `.navbar-toggler-icon` span and toggles the `.show` class on the
collapsible region.

<Example code={`<bs-navbar background="body-tertiary" expand="never">
  <span slot="brand">Hidden brand</span>
  <bs-nav>
    <bs-nav-item href="#" active>Home</bs-nav-item>
    <bs-nav-item href="#">Link</bs-nav-item>
  </bs-nav>
</bs-navbar>`} />

## API

| Attribute     | Type                                                               | Default   | Notes                                                                                                            |
| ------------- | ------------------------------------------------------------------ | --------- | ---------------------------------------------------------------------------------------------------------------- |
| `expand`      | `sm` \| `md` \| `lg` \| `xl` \| `xxl` \| `always` \| `never`       | `lg`      | Breakpoint at which the navbar becomes horizontal.                                                               |
| `theme`       | `light` \| `dark` \| `auto`                                        | `auto`    | Sets `data-bs-theme` on the inner `<nav>`. `auto` inherits.                                                      |
| `background`  | `primary` \| `dark` \| `body-tertiary` \| ...                      | `—`       | Applies `bg-{value}`.                                                                                            |
| `placement`   | `static` \| `fixed-top` \| `fixed-bottom` \| `sticky-top` \| `sticky-bottom` | `static`  | Pins the navbar to the viewport.                                                                                 |
| `container`   | `fluid` \| `sm` \| `md` \| `lg` \| `xl` \| `xxl` \| `none`         | `fluid`   | Inner wrapper. `none` skips it so you can nest inside your own container.                                        |
| `fluid`       | `boolean`                                                          | `true`    | Legacy shorthand — `fluid=false` is equivalent to `container="sm"`-ish `.container`.                              |
| `fixed`       | `boolean`                                                          | `false`   | Legacy shorthand for `placement="fixed-top"`.                                                                    |
| `sticky`      | `boolean`                                                          | `false`   | Legacy shorthand for `placement="sticky-top"`.                                                                   |
| `brand-href`  | `string`                                                           | `#`       | Anchor target for the brand slot wrapper.                                                                        |
| `expanded`    | `boolean` (reflected)                                              | `false`   | Current toggler state. Toggled by clicks on the built-in toggler.                                                |

| Slot      | Purpose                                                        |
| --------- | -------------------------------------------------------------- |
| (default) | Main content — rendered inside the collapsible region.         |
| `brand`   | Brand area; wrapped in `<a class="navbar-brand">`.             |
| `end`     | Trailing content (e.g. a search form) after the default slot.  |

See `packages/components/src/navbar/` in the source for the complete
property / event list.
