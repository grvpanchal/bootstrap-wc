## Flush

## Install

```sh
# as an npm package
npm install @bootstrap-wc/components
# or copy the source
npx bwc add accordion
```

## Usage

```ts
import '@bootstrap-wc/components/accordion';
```

```html
<bs-accordion>
  <bs-accordion-item heading="Item #1" open>First panel content.</bs-accordion-item>
  <bs-accordion-item heading="Item #2">Second panel content.</bs-accordion-item>
</bs-accordion>
```

## API

### `bs-accordion`

#### Attributes / properties

| Name | Type | Default | Description |
|---|---|---|---|
| `flush` | `boolean` | `false` | Render edge-to-edge without outer borders / rounded corners. |
| `always-open` | `boolean` | `false` | Allow multiple items to be expanded simultaneously. |

### `bs-accordion-item`

#### Attributes / properties

| Name | Type | Default | Description |
|---|---|---|---|
| `open` | `boolean` | `false` | Whether the panel is expanded (reflected). |
| `heading` | `string` | — | Convenience label for the collapsed-state button. Use the `header` slot for rich content. |

#### Slots

| Slot | Description |
|---|---|
| _(default)_ | Panel body content. |
| `header` | Button label (use instead of `heading` for rich content). |

#### Events

| Event | Description |
|---|---|
| `bs-accordion-item-toggle` | Fires on state change. `detail: { open: boolean }`. Bubbles. |
