Dropdowns use [@floating-ui/dom](https://floating-ui.com) for placement. They flip to stay in the viewport, shift to avoid overflow, and close on outside click or ESC.

## Examples

### Single button

A `<bs-dropdown>` with a button trigger. Use any Bootstrap color via `variant`.

<Example code={`<bs-dropdown label="Dropdown button" variant="secondary">
  <bs-dropdown-item slot="menu" href="#">Action</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Another action</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Something else here</bs-dropdown-item>
</bs-dropdown>`} />

### Single button

Render the toggle as an anchor (`<a role="button">`) via `toggle-tag="a"`.

<Example code={`<bs-dropdown toggle-tag="a" label="Dropdown link" variant="secondary">

  <bs-dropdown-item slot="menu" href="#">Another action</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Something else here</bs-dropdown-item>
</bs-dropdown>`} />

## Menu items

Items accept `href` (renders as an anchor) or `as="button"` (renders as a `<button>`).

<Example code={`<bs-dropdown label="Dropdown">
  <bs-dropdown-item slot="menu" as="button">Action</bs-dropdown-item>
  <bs-dropdown-item slot="menu" as="button">Another action</bs-dropdown-item>
  <bs-dropdown-item slot="menu" as="button">Something else here</bs-dropdown-item>
</bs-dropdown>`} />

Use `text` to render non-interactive text (as `<span class="dropdown-item-text">`).

<Example code={`<bs-dropdown open label="Menu">
  <bs-dropdown-item slot="menu" text>Dropdown item text</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Action</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Another action</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Something else here</bs-dropdown-item>
</bs-dropdown>`} />

### Active

Mark an item as `active` — it also sets `aria-current="true"`.

<Example code={`<bs-dropdown open label="Menu">
  <bs-dropdown-item slot="menu" href="#">Regular link</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#" active>Active link</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Another link</bs-dropdown-item>
</bs-dropdown>`} />

<Example code={`<bs-dropdown open label="Menu">

## Dropdown options

### Auto close behavior

- Default (`auto-close` is true): close on click outside, click on item, or ESC.
- Disable auto-close entirely with `auto-close="false"` to require manual close via code or ESC.

<Example code={`<bs-dropdown label="Default dropdown">
  <bs-dropdown-item slot="menu" href="#">Menu item</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Menu item</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Menu item</bs-dropdown-item>
</bs-dropdown>
<bs-dropdown auto-close="false" label="Manual close">
  <bs-dropdown-item slot="menu" href="#">Menu item</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Menu item</bs-dropdown-item>
  <bs-dropdown-item slot="menu" href="#">Menu item</bs-dropdown-item>
</bs-dropdown>`} />

## Placement

All [floating-ui placements](https://floating-ui.com/docs/computePosition#placement) are supported via the `placement` attribute; this overrides `drop`.

```html
<bs-dropdown placement="bottom-start">…</bs-dropdown>
<bs-dropdown placement="top-end">…</bs-dropdown>
<bs-dropdown placement="right-start">…</bs-dropdown>
```

## API

### `<bs-dropdown>` attributes / properties

| Name | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | `'Dropdown'` | Trigger label. Overridden by the `label` slot. |
| `variant` | Bootstrap variant | `secondary` | Trigger button variant. |
| `size` | `sm` \| `md` \| `lg` | — | Trigger size modifier. |
| `placement` | Floating-UI placement | — | Explicit placement (overrides `drop`). |
| `drop` | `down` \| `up` \| `end` \| `start` \| `center` \| `up-center` | `down` | Wrapper direction modifier. |
| `open` | `boolean` | `false` | Menu open state. |
| `no-caret` | `boolean` | `false` | Hide the caret indicator (non-split only). |
| `auto-close` | `boolean` | `true` | Close on outside click / item select. |
| `split` | `boolean` | `false` | Render as a split button. |
| `menu-end` | `boolean` | `false` | Right-align the menu (`.dropdown-menu-end`). |
| `menu-dark` | `boolean` | `false` | Dark variant menu (`.dropdown-menu-dark`). |
| `toggle-tag` | `button` \| `a` | `button` | Render the toggle as `<button>` or `<a>`. |

### Events

| Event | Description |
|---|---|
| `bs-show` / `bs-shown` | Opening lifecycle. |
| `bs-hide` / `bs-hidden` | Closing lifecycle. |

### `<bs-dropdown-item>` attributes

| Name | Type | Description |
|---|---|---|
| `href` | `string` | Renders as a link. |
| `as` | `a` \| `button` | Underlying element (default `a`). |
| `disabled` | `boolean` | Disable the item. |
| `active` | `boolean` | Mark as active (also sets `aria-current="true"`). |
| `divider` | `boolean` | Render as a divider `<hr>`. |
| `header` | `boolean` | Render as a menu header. |
| `text` | `boolean` | Render as non-interactive item text. |

## Install

```sh
npm install @bootstrap-wc/components @floating-ui/dom
# or
npx bwc add dropdown
```
